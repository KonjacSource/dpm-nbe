## DPM-NBE

Experiments for implementing Dependent Pattern Matching under Normalization by Evaluation setup.

### Syntax

Example

```agda
let trans : (A : U) (a b c : A) -> Id A a b -> Id A b c -> Id A a c =
  λ A a b c refl refl . refl; 

let cong : (A B : U) (f : A -> B) (a b : A) -> Id A a b -> Id B (f a) (f b) =
  λ A B f a b refl . refl;

let H : (m n : Nat) -> Id Nat n (plus m zero) -> Id Nat m zero -> Id Nat n zero =
  λ m n refl refl . refl;

let C : (A : U) -> Id Nat (suc zero) zero -> A = 
  λ A (!); -- Absurd pattern

U
```

Note.

- ```plus```, ```suc```, ```Id``` must be full applied.
- For simplicity, we only have ```refl``` , absurd pattern ```(!)``` and variables as patterns, ```zero``` and ```suc``` is not a pattern.
  It is just to show DEPENDENT pattern matching.
- There are some issues with pretty printing, I'll fix them when I want to.
